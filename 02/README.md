## ★介绍

从基础到实战，详细讲解跨平台应用开发的方方面面，包含 Uniapp 开发常用知识点，基础 api，前端交互、组件封装，后端 Nodejs 开发、前后端联调和调优部署，是一套非常全面的综合课程。

- 目的：想独立开发小程序？或者是开发一款 app？
- 目标：实现一个某易音乐项目的开发

本小册由三部分组成：

基础--Uniapp 快速入门
实战--Uniapp 进阶，音乐项目实战和后端 Nodejs 开发
福利-- 优化，避坑和经验分享
